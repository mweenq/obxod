# obxod

Локальный SOCKS5-прокси для обхода блокировок и замедления Telegram

### Основные возможности

- SOCKS5 сервер на 127.0.0.1:1080 (или любой другой)
- Автоматическое определение Telegram-трафика
- Режимы:
  - `auto` - оптимален для большинства блокировок
  - `ws` - обход через web-socket
  - `direct` - без обхода
- WebSocket-маскировка под web.telegram.org (kws1–kws5)
- Поддержка UDP (голосовые звонки, видеозвонки, нужно включить p2p соединение в конфиденциальность → звонки → peer-to-peer)
- Возможность поставить в автозапуск через `settings.bat`

### Быстрый старт

1. Запустите .bat

2. В Telegram → Настройки → Данные и память → Прокси  
   → Добавить прокси → SOCKS5  
   → Сервер: 127.0.0.1  
   → Порт: 1080  

3. Если пишет  
   **«Прокси неправильно настроен, перезапустите прокси»**  (или что-то подобное, у меня на английском)
   → перезапустите прокси
   → при сохранении ошибки, попробйте перезапустить Telegram

### Полезные флаги

```bash
obxod -p 9050              # другой порт
obxod -b 0.0.0.0 -p 1080   # слушать на всех интерфейсах (для всех устройств вашей сети, не проверялось)
obxod -m ws                # всегда использовать WebSocket (самый надёжный обход)
obxod -m direct            # отключить обход, чистый TCP
obxod --dc 2:149.154.167.51  # задать свой IP для DC2
obxod --logs True          # включить подробные логи в консоль
